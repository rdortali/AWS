Welcome to TIBCO AWS Demo Portal


Make sure that you logged on Amazonn AWS.
By clicking the launchStack button you will be provided with some options to change some properties.
The only properties you need to change are:
- number of instances you would like to run,
- your KeyName,
- SSHLocation (which is your IPaddress).
After inserting these properties you can create stack which will create the number off instances you selected.
